# Task 2: Phishing Awareness Campaign

## Project Overview
This project was developed during my cybersecurity internship at CodeAlpha. The goal is to design an educational campaign to train users to identify and defend against modern phishing methodologies.

## Campaign Components
1. **Scenario 1:** Urgent Account Suspension Phishing (Targeting Credential Harvesting).
2. **Scenario 2:** Missed Package Delivery Scam (Targeting Financial/Card Data).
3. **Educational Material:** A 4-step Phishing Prevention Cheat Sheet for end-users.
